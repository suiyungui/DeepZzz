```ini
Environment=HOST=0.0.0.0
Environment=PORT=7860

Environment=POSE_PROVIDER=CPUExecutionProvider
Environment=ANALYSIS_WIDTH=320
Environment=ANALYSIS_FPS=1

Environment=AUDIO_DEVICE=plughw:0,0

```

## 迁移到新板子的要点

1. 主工程复制到新板：`/home/z/deepzzz_k2_edge`
2. 安装依赖：参考 `deepzzz_k2_edge/requirements-board.txt`
3. 安装自启服务：

```bash
sudo cp /home/z/deepzzz_k2_edge/scripts/deepzzz-k2-edge.service /etc/systemd/system/deepzzz-k2-edge.service
sudo systemctl daemon-reload
sudo systemctl enable --now deepzzz-k2-edge.service
```

4. 旧摄像头服务不要同时启用：

```bash
sudo systemctl disable --now camera-hls.service camera-web.service
```

5. 是否启用 `day-night-sync.service` 取决于新板是否还需要 IR-CUT 联动。
